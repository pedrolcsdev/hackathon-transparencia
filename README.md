# Portal da Transparencia

Projeto de Hackathon do Portal da Transparencia do Governo do Maranhao, feito com HTML, CSS e JavaScript.

## Estrutura

- `index.html`: pagina inicial.
- `despesas.html`, `receitas.html`, `recursos-humanos.html`, `contratos-licitacoes.html`: paginas de consulta.
- `paginas-consulta.css` e `paginas-consulta.js`: estilos e scripts das paginas internas.
- `images/`: imagens principais do portal.
- `imagens/icones/`: icones usados na interface.
